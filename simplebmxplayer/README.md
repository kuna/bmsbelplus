# SimpleBmxPlayer

### abstract
- just plays \*.bmx(bms, bme, bml, etc.) file.
- good to use when editing \*.bmx files.

### dependency
- ```SDL```
- ```SDL_image```
- ```SDL_mixer```
- ```SDL_ttf```
- ```SDL_FontCache```
- ```Zeranoe FFmpeg```
- ```Soundtouch```
- ```bmsbel (libiconv)```

### how to use
- drag BMS file to program.
